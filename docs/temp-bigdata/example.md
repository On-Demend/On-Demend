## BigData 1-2
- Environment : **hst**

- Assignment : 시각화 분석 | 90m

### 요구사항

1. 검색사이트: (https://www.mois.go.kr)의 정책자료 → 통계 → **주민등록 인구통계 연령별 인구현황에서 통계표 탭 선택** - 검색한 후에 결과가 나오면, 전체읍면동현황을 선택하고 csv 파일 다운로드 버튼을 누름
    - 검색조건: 기간 2020년01월 ~ 2020년01월

    - 구분: 남여구분

    - 연령구분 단위: 1세

    - 만 연령구분 0 - 100이상

    - 지역: 전국시군구현황 [검색] -> 전체읍면동현황 csv 파일 다운로드

    - 저장파일명: hs2.csv 

2. 특정 행정구역명 `구미시 산동읍`을 입력으로 받아서 남자의 연령별 인구수를 수평막대그래프로 출력하시오. - 타이틀은 구미시 산동읍 남자 인구분표
    - 범례는 좌측하단, 색상은 남자는 red

    - x축 이름은 인구수, y축 이름은 연령

3. 특정 행정구역명 “구미시 산동읍”을 입력으로 받아서 여자의 연령별 인구수를 수평막대그래프로 출력하시오. - 타이틀은 구미시 산동읍 여자 인구분표
    - 범례는 우측상단, 색상은 여자는 blue로 설정

    - x축 이름은 인구수, y축 이름은 연령

4. 특정 행정구역명 “구미시”을 입력으로 받아서 남녀 성별 인구분포를 항아리모양 그래프로 출력하시오. - 타이틀은 구미시 남녀 인구분표
    - 범례는 우측하단, 색상은 남자는 red, 여자는 blue로 설정

    - x축 이름은 인구수, y축 이름은 연령

    - 남자의 연령별 인구수를 x축의 –쪽에 출력하고, -부호가 깨어지면 안됨

5. 특정 행정구역명 “서울특별시”을 입력으로 받아서 남녀 성별 인구분포를 항아리모양그래프로 출력하시오. - 타이틀은 서울시 남녀 인구분표

    - 범례는 좌측상단에 위치, 색상은 남자는 red, 여자는 blue로 설정

    - x축 이름은 인구수, y축 이름은 연령

    - 남자의 연령별 인구수를 x축의 –쪽에 출력하고, -부호가 깨어지면 안됨
